# Main-Project-2_Walmart-Sales-Analysis-using-Excel
This project encompasses comprehensive sales data for Walmart across various store locations, spanning the fiscal years 2010 through 2012.
Data cleaning completed to make the data reliable for analysis.
Checked spellings
Checked for duplicates
Created pivot tables
Created dashboard
Analyzed sales data for the top-performing stores from 2010 to 2012 and observed a decline in sales during that period.
